# 📊 Power BI Learning Journey

## Overview

Over the course of this program, I built a comprehensive set of skills in Microsoft Power BI, progressing from foundational data preparation through to advanced modelling, visualisation, and reporting. The journey covered the full analytics workflow — from raw data to decision-ready dashboards — grounded in real business scenarios including Adventure Works and Tailwind Traders.

---

## 🔧 Data Preparation & Power Query

- Connected to and cleaned multiple data sources including Excel, SQL Server, SharePoint, and Python scripts
- Applied transformations such as filtering, removing duplicates, splitting columns, and changing data types
- Profiled data using Column Quality, Column Distribution, and Column Profile to detect anomalies and errors
- Merged and appended tables to produce unified, analysis-ready datasets
- Used `TRIM()`, `PROPER()`, `UPPER()`, and `LOWER()` to standardise text data

---

## 🗂️ Data Modelling

- Built relational data models in Power BI Desktop defining one-to-many, many-to-many, and one-to-one relationships
- Configured storage modes — Import, DirectQuery, and Dual — to optimise performance
- Assigned correct data types across all columns including currency, whole number, decimal, date, and text
- Created calculated columns, measures, and tables using DAX
- Built Calendar tables using both DAX and M language for time intelligence calculations
- Configured aggregated tables to improve DirectQuery model performance
- Implemented Role-Level Security (RLS) to restrict data access by role

---

## 📐 DAX

- Wrote measures using `SUM`, `CALCULATE`, `FILTER`, `SUMX`, `IF`, `LASTDATE`, `LASTNONBLANK`, and `USERELATIONSHIP`
- Used `SUMIFS` logic and context modification to produce conditional calculations
- Applied time intelligence functions for year-over-year comparisons and running totals
- Created quick measures and referenced existing measures within new calculations
- Built parent-child hierarchy logic using the `PATH` DAX function

---

## 📈 Visualisation & Reporting

- Created column charts, bar charts, line charts, donut charts, treemaps, ribbon charts, scatter plots, waterfall charts, KPI cards, and Q&A visuals
- Applied conditional formatting, tooltips, slicers, bookmarks, and page navigation buttons
- Used the Analytics pane to add forecasting, trend lines, and symmetry shading
- Configured report themes, font sizes, and visual borders for a consistent, professional layout
- Optimised reports for mobile using the Mobile Layout editor in both Power BI Desktop and Service

---

## 🖥️ Power BI Service & Administration

- Published reports to Power BI Service and pinned visuals to executive dashboards
- Configured workspace roles — Viewer, Contributor, Member, and Admin
- Set up scheduled refresh using personal and on-premises data gateways
- Applied sensitivity labels to datasets, dataflows, and paginated reports
- Used Lineage View to trace data flow from source to report
- Explored deployment pipelines for promoting content across development and production environments

---

## 🤖 Generative AI & Copilot

- Explored Microsoft Copilot's role in Power BI for report generation and DAX assistance
- Used Bing Copilot to troubleshoot and validate DAX formulas
- Assessed the risks and limitations of using generative AI tools in a data analytics context

---

## 🎓 Certification Preparation

- Completed full preparation for the **Microsoft PL-300: Power BI Data Analyst** certification exam
- Practiced across all core exam domains: data preparation, data modelling, visualisation, and analytics
- Achieved a 82% score on the mock exam with identified areas for further review
